## Cadastro de Alunos
### O projeto se trata de um crud feito pelo Visual Studio, com 3 interfaces:
#### Login
#### Esqueci minha senha
#### Cadastrar alunos

#### O login valida se as credenciais digitadas tem acesso e caso sim ele abre a interface de cadastro de alunos

![image](https://github.com/user-attachments/assets/62c647e7-4c6c-4b0c-b531-59f36a10d26d)


### Cadastro de Alunos

#### O cadastro tem as seguintes funcionalidades:

#### -Incluir aluno
#### -Atualizar aluno com matrícula já existente
#### -Consultar aluno por matrícula
#### -Consultar todos os alunos cadastrados
#### -Limpar os campos preenchidos
#### -Remover por matrícula
#### -Remover pela linha seleciona no datagridview
